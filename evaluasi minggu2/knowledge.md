yang saya pahami selama saya belajar javascript di bulan ke 2 di minggu ke 2 ini adalah :

*berikut perincian nya :*
- pop : menghapus data di dalam array
- unshift : menambahkan elemen di awal array
- shift : menghapus dan mengembalikan elemen dalam array
- indexof : mencari elemen dalam array
- push : menambahkan data dalam array
- find : mencari elemen dalam array
- reduce : menggabungkan data dalam array
- concat : menaggabungkan elemen dalam array
- map : membuat array baru dengan nilai yg baru
- reverse : membalikan data yg awal mula nya dari dari awal ke akhir dan dirubah dari akhir ke awal
- sort : mensortir isi data dalam array dari huruf kecil ke besar, dari huruf a-z
- filter : membuat array baru dengan melihat kondisi
- spread : menggabungkan atau menyalin data array yg banyak
