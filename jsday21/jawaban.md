## Jawaban soal
1. fungsi dari localstorage dalam javascript adalah menyimpan data di dalam web dan mudah digunakan, cepat di akses dan lebih besar kapasitas nya daripada cookie
2. perbedaan localstorage dan session storage adalah jika localstorage itu bisa menyimpan dalam jangka waktu lama meski web di tutup kalau session storage menyimpan hanya sementara jika web nya di tutup maka data tersebut hilang
3. caranya kita buat function setelah itu membuat    ```localStorage.setItem("key", "value");``` kemudian alert
4.  caranya buat function setelah itu kita buat ```localStorage.getItem("key");``` kemudian ```document.getElementById()```
5.  caranya kita buat function setelah itu membuat    ```localStorage.removeItem("key");``` kemudian alert
